# Lystak

Agnostic task list (can be used as a shopping list), open-source, collaborative and self-hosted 

## Developement

Requirements:
- Make
- Docker with docker-compose

### Install dependencies

```bash
make init
```

### Run dev server

```bash
make run
```

### Dev server URL

- [front](http://localhost:3200/)
- [admin](http://localhost:3201/)
